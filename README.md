# Smart Mission Management System

A Java-based console application for managing space mission records.

## Features

- Add Mission
- View All Missions
- Search Mission by ID
- Update Mission Status
- Delete Mission
- File-based Data Persistence
- Input Validation
- Exception Handling

## Technologies Used

- Java
- OOP Concepts
- Collections Framework (ArrayList)
- File Handling
- Exception Handling

## Project Structure

- Mission.java – Model Class
- MissionService.java – Business Logic & File Handling
- Main.java – Menu-Driven Interface
- missions.txt – Persistent Storage

## How to Run

1. Compile all Java files
2. Run Main.java
3. Use the menu options to manage missions
